# Venta de Tiquetes - VuelosCR.SA

Proyecto de escritorio en C# (.NET Framework 4.8) para la gestión y venta de tiquetes aéreos.

## Descripción
Esta aplicación permite registrar, calcular tarifas y gestionar la venta de tiquetes de vuelos, mostrando información relevante y permitiendo la administración de los registros.

## Características
- Registro de clientes y vuelos
- Cálculo automático de tarifas, impuestos y servicios
- Visualización y gestión de tiquetes en una tabla
- Eliminación y limpieza de registros
- Interfaz intuitiva y moderna

## Tecnologías utilizadas
- C# 7.3
- Windows Forms
- .NET Framework 4.8

## Cómo ejecutar
1. Clona este repositorio.
2. Abre la solución en Visual Studio.
3. Compila y ejecuta el proyecto.
